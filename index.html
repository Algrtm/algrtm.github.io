Artworks

Artworks are the abstract visualizations of the algorithms and mathematical equations. Since those are not bound by a physical canvas, they can be rendered in any size. Maximum available yet is approximately 1 gigapixel (32,768 x 32,768 pixels). View more of my art in the gallery section. (click on one to view fullscreen)

About the project

This project began as a merger of my hobbies: programming and digital art. The result - high resolution abstract artworks, based on algorithms and math. I post new artworks on my Instagram page. You can find more in the contacts section.

How I create

For the longest time I have been using Java as my programming language of choice, so I wrote a software that can render any algorithm and/or complex math equations as a still image or frame by frame animation.
